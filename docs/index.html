    elsass

    @mixin elsass($media: from $ELSASS-BPT-MIN, $width, $gutter, $offset, $extra) { ... }
    View source

    Description

    Builds any responsive layout item.
    Uses global settings before to fallback to responsive ones.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $media

    Defines the related media-query(ies). See elsass-media mixin.

    		Listfrom $ELSASS-BPT-MIN
    $width

    Defines the element width and (optionally) max-witdh from the following values.

    • 3: divises the global columns number (12 by default) per 3;
    • 3 of 6: applies a width of 50%;
    • 'max': applies a width value of 100% and a max width related to the settings.
    		List— none
    $gutter

    Applies gutters as padding and/or margin based on the following valid values.

    • A map, as defined in $elsass-gutters;
    • A shorthand as 'default', 'none' or an $elsass-behaviours key, optionally preceded by an $elsass-gutters key to override the default gutter value.
      Gutter behaviour related keywords accept suffixes to alterate defined gutter sides. Valid sufixes are '-top', '-right', '-bottom', '-left', '-x' and '-y'. If no suffix is provided all gutter related sides will be alterate.
      $gutter: 'all-over'; // or
$gutter: 'nest'; // or
$gutter: 'all-over' 'nest';
    		List or Map— none
    $offset

    Alterates the element and/or content position by pushing, pulling or centering it via margin and/or padding through the following values.

    • A map of 'in' and/or 'out' offset values as:
      • a map of sides related <length> or container width ratios;
        $offset: ('out': ('left': 3));
    • A shorthand using one or more $elsass-offsets key, optionally followed by a <length> or container width ratio. If no ratio is provided auto will be used.
        $gutter: 'center' 'start' 3;
    		List— none
    $extra

    List of extra CSS rules to apply. Any gutter or width related value can be indirectly set via elsass-gutter or elsass-width functions by providing a map with gutter or width as key and where the value is the related argument list.

    		List— none

    Example

    SCSS

    .foo {
  border-style: solid;
  @include elsass(from 'l', 3, 'all-out', 'before' 3, (border-width: (gutter: .25)));
}

    Output

    Arguments and @content relative CSS wrapped into tidy media-queries.

    elsass-width

    @function elsass-width($width, $gutter-width) { ... }
    View source

    Description

    Calculate width by substracting potential right and left gutters used as margin.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $width

    Ratio of the container width or <length>.
    Enter 'max' or a breakpoint name to use the global/responsive container max width defined through $elsass.

    		Number or String— none
    $gutter-width

    Sum or list of the right and left gutter ratios or lengths.
    A one item long map using a breakpoint name as key can be provided to use this breakpoint related gutter length as the ratio base.
    A single breakpoint can be used as a shorthand of ('breakpoint': 1).

    		Number or List or Map— none

    Returns

    Number —

    <length> (via calc() if needed).

    Example

    SCSS

    width: elsass-width(3, 'm');
// = width: elsass-width(1 of 4, ('m': 1));

    CSS output

    // Assuming 'm' breakpoint gutter length is set to 30px through $elsass.
width: calc(25% - 30px);
// Assuming 'm' breakpoint gutter length is set to 5% through $elsass.
width: 20%;

    elsass-gutter

    @function elsass-gutter($gutter, $offset) { ... }

    Description

    Calculate a gutter length.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $gutter

    Gutter ratio or length.
    A one item long map using a breakpoint name as key can be provided to use this breakpoint related gutter length as the ratio base.
    A single breakpoint can be used as a shorthand of ('breakpoint': 1).

    		Number or String or Map— none
    $offset

    Ratio the container width or <length> to add or substract;

    		Number or Length— none

    Returns

    Number —

    <length>

    Example

    SCSS

    margin-right: elsass-gutter(('m': 0.5));
margin-left: elsass-gutter(('m': 0.5), 3);

    CSS output

    // Assuming 'm' breakpoint gutter length is set to 30px through $elsass.
margin-right: 15px;
margin-left: calc(15px + 25%);
// Assuming 'm' breakpoint gutter length is set to 5% through $elsass.
margin-right: 2.5%;
margin-left: 27.5%;
